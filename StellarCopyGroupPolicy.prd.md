# AIpromptStellarAPI

## Create a Powershell 5.1 script that:
1. expects 2 parameters: SourceAgentGroup and DestinationAgentGroup. 
2. extracts the policy of the StellarOne SourceAgentGroup
3. adds it to the DestinationAgentGroup.  
3. Creates the destination group if it does not exist.

## Use the following files:
1. StellarOne3.3.1392.openApi.json for the OpenAPI descriptions 
2. StellarOneAPI.conf: StellarOne URL 
3. secrets.txt: API key  

## Context:
Your output will be used as an example on how to use the StellarOne API in a training; make your outputs descriptive for non-technical people so they can learn from it

## General Instructions:
1. Make an MD file that describes what the PS1 script does
2. Add the file creation date and time as a comment in every file you make
3. Ask for more details when needed; do not guess
4. Test your script before delivering it
